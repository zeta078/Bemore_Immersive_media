# BMO 프로젝트 실행 안내

## 1. 프로젝트 개요

BMO는 Raspberry Pi 5 기반으로 제작한 생활형 음성 AI 에이전트 프로토타입입니다.
사용자는 음성으로 BMO와 대화할 수 있으며, 냉장고 재료 확인, 레시피 추천, 날씨 조회, 카메라 기능, 미니게임 등의 기능을 사용할 수 있습니다.

본 프로젝트는 로컬 LLM(Ollama), STT, TTS, GUI, SQLite 데이터베이스, 외부 백엔드 서버를 함께 사용하는 구조입니다.

---

## 2. 실행 전 참고 사항

본 프로젝트는 다음 환경을 전제로 개발되었습니다.

* Raspberry Pi 5
* Python 3.x
* 마이크
* 스피커
* 카메라
* Ollama 로컬 LLM
* SQLite DB 파일
* OpenWeatherMap API Key
* 외부 백엔드 서버

일반 PC 환경에서도 일부 코드는 실행할 수 있으나, 하드웨어 및 서버 연결 상태에 따라 일부 기능이 제한될 수 있습니다.

제한될 수 있는 기능은 다음과 같습니다.

* 마이크 입력 및 음성 인식
* TTS 음성 출력
* 카메라 촬영 및 이미지 분석
* 외부 백엔드 기반 음식 추천
* 영수증 OCR
* 날씨 조회
* 냉장고 및 레시피 DB 기능

실행 환경 차이로 직접 실행이 어려운 경우, 함께 제출한 시연 영상을 통해 주요 기능 동작을 확인할 수 있습니다.

## 4. 설치 방법

프로젝트 코드 폴더로 이동한 뒤, 필요한 Python 패키지를 설치합니다.

```bash
cd BMO
pip install -r requirements.txt
```

## 5. Ollama 실행

BMO의 일반 대화 기능은 로컬 Ollama 서버를 사용합니다.
별도 터미널에서 Ollama 서버를 실행합니다.

```bash
ollama serve
```

프로젝트에서 사용하는 모델이 설치되어 있지 않은 경우, 아래와 같이 모델을 설치합니다.

```bash
ollama pull gemma4:e2b-it-qat
```

※ 실제 사용 모델명은 `BMO/bmoAPI.py` 또는 설정 파일에 지정된 모델명을 기준으로 확인할 수 있습니다.

## 6. 환경 변수 설정

본 프로젝트는 일부 기능에서 외부 서버 주소와 API Key를 사용합니다.
제출본에는 보안상 API Key, 비밀번호 등 민감 정보가 포함되어 있지 않습니다.

### 6-1. 외부 백엔드 서버 주소 설정

외부 백엔드 서버 주소는 환경 변수로 설정할 수 있습니다.

#### Windows CMD

```cmd
set BMO_BACKEND_URL=http://백엔드_서버_IP:포트
```

#### Windows PowerShell

```powershell
$env:BMO_BACKEND_URL="http://백엔드_서버_IP:포트"
```

#### Linux / Raspberry Pi

```bash
export BMO_BACKEND_URL="http://백엔드_서버_IP:포트"
```

기본 백엔드 서버 주소가 코드에 설정되어 있더라도, 서버가 실행 중이지 않으면 음식 추천, 이미지 분석, OCR, 영양 정보 조회 등 일부 기능은 동작하지 않을 수 있습니다.

---

### 6-2. 날씨 API Key 설정

날씨 조회 기능은 OpenWeatherMap API를 사용합니다.
제출본에는 보안상 OpenWeatherMap API Key를 포함하지 않았으므로, 날씨 기능을 사용하려면 OpenWeatherMap에서 API Key를 별도로 발급받아야 합니다.

발급받은 API Key는 환경 변수로 설정할 수 있습니다.

#### Windows CMD

```cmd
set OPENWEATHER_API_KEY=발급받은_API_KEY
```

#### Windows PowerShell

```powershell
$env:OPENWEATHER_API_KEY="발급받은_API_KEY"
```

#### Linux / Raspberry Pi

```bash
export OPENWEATHER_API_KEY="발급받은_API_KEY"
```

프로젝트의 날씨 설정 파일은 다음과 같이 환경 변수 값을 읽어오도록 구성되어야 합니다.

```python
# BMO/features/weather/weather_config.py

import os

OPENWEATHER_API_KEY = os.getenv("OPENWEATHER_API_KEY", "")
```

API Key가 설정되어 있지 않으면 날씨 조회 기능은 정상적으로 동작하지 않을 수 있습니다.

## 7. 실행 방법

프로젝트 코드 폴더인 `BMO` 폴더 안에서 실행합니다.

```bash
cd BMO
python main.py
```

Raspberry Pi에서 GUI 화면 출력이 필요한 경우, 다음과 같이 실행할 수 있습니다.

```bash
cd BMO
DISPLAY=:0 python main.py
```

## 9. 종료 방법

터미널에서 실행 중인 경우 다음 키로 종료할 수 있습니다.


```bash

```

## 10. 실행 시 주의사항

* Ollama 서버가 실행 중이지 않으면 일반 대화 기능이 제한될 수 있습니다.
* 지정된 Ollama 모델이 설치되어 있지 않으면 LLM 응답 기능이 동작하지 않을 수 있습니다.
* 마이크 또는 스피커가 연결되어 있지 않으면 음성 입출력 기능이 정상적으로 동작하지 않을 수 있습니다.
* 카메라가 연결되어 있지 않으면 카메라 촬영 및 이미지 분석 기능이 제한됩니다.
* OpenWeatherMap API Key가 설정되어 있지 않으면 날씨 조회 기능이 제한됩니다.
* 외부 백엔드 서버가 실행 중이지 않으면 음식 추천, 이미지 분석, OCR, 영양 정보 조회 기능이 제한될 수 있습니다.
* DB 파일이 없거나 초기 데이터가 없으면 냉장고 및 레시피 기능이 정상적으로 표시되지 않을 수 있습니다.
* 제출본에는 개인정보 보호를 위해 API Key, 비밀번호 등 민감 정보는 포함하지 않습니다.
